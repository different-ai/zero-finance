# 0 Finance Docs

This directory contains the public documentation for 0 Finance, powered by Mintlify and deployed to https://docs.0.finance.

## Local preview

Install the Mintlify CLI:

```bash
npm i -g mint
```

From the `packages/docs` directory, run:

```bash
mint dev
```

Preview at `http://localhost:3000`.

## Updating docs

- Edit `.mdx` files inside `packages/docs/`
- Update navigation in `packages/docs/docs.json`
- Push changes to `main` to deploy automatically

## Assets

The docs use the 0 Finance logo hosted at `https://0.finance/new-logo-bluer.png`.
