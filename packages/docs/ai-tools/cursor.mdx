Use the 0 Finance MCP server to let Cursor call financial tools like balance lookups, invoice creation, and transfer proposals.

## MCP endpoint

- URL: `https://www.0.finance/api/mcp`
- Auth: `Authorization: Bearer <api_key>`

Create an API key in **Settings → API Keys** before connecting.

## Configure Cursor

Add a remote MCP server in Cursor’s MCP settings with the URL and auth header above.

Example configuration (adapt to Cursor’s MCP settings format):

```json
{
  "mcpServers": {
    "0-finance": {
      "url": "https://www.0.finance/api/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
```

## Available tools

- `list_saved_bank_accounts`
- `get_balance`
- `propose_bank_transfer`
- `list_proposals`
- `create_bank_account`
- `create_invoice`, `update_invoice`, `list_invoices`, `get_invoice`, `send_invoice`
- `list_transactions`, `get_transaction`
- `attach_document`, `list_attachments`, `remove_attachment`
- `get_payment_details`, `share_payment_details`
- `dismiss_proposal`

<Info>
  Transfers are always proposed first and require user approval in the
  dashboard.
</Info>
