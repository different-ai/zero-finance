## GET `/api/health`

Use this endpoint for uptime checks and local verification.

### Request

```bash
curl https://0.finance/api/health
```

### Response

```json
{
  "status": "ok",
  "timestamp": "2026-01-12T00:00:00.000Z",
  "message": "Server is running"
}
```
