## Install with Bun

```bash
bun add -g agent-bank
```

Verify the installation:

```bash
finance --version
```

<Tip>
  The CLI binary is `finance`. You can also use the shorter aliases `0` or
  `zero`.
</Tip>

---

## Run from the repo

The CLI lives in `packages/cli` and can be run directly from source:

```bash
bun install
bun --cwd packages/cli run dev -- auth whoami
```

---

## Authenticate

### Step 1: Get your API key

1. Log in to [0.finance](https://0.finance)
2. Go to **Settings â†’ API Keys**
3. Click **Create API Key**
4. Copy the key (it starts with `sk_live_` or `sk_test_`)

<Warning>
  Store your API key securely. It provides full access to your workspace.
</Warning>

### Step 2: Log in

```bash
finance auth login --api-key sk_live_your_api_key_here
```

This stores your credentials in `~/.zero-finance/config.json`.

### Step 3: Verify

```bash
finance auth whoami
```

```json
{
  "workspace_id": "ws_xxx",
  "workspace_name": "My Company",
  "key_id": "key_xxx",
  "key_name": "Production Key",
  "align_customer_id": "align_xxx"
}
```

---

## Configuration

The CLI stores configuration in `~/.zero-finance/config.json`:

```json
{
  "apiKey": "sk_live_xxx",
  "baseUrl": "https://0.finance"
}
```

### Custom base URL

For self-hosted deployments or local development:

```bash
finance auth login --api-key sk_test_xxx --base-url http://localhost:3050
```

---

## Log out

Remove stored credentials:

```bash
finance auth logout
```
